= Query plan adapters

include::ROOT:partial$version-check.adoc[]

The Cerbos xref:api:index.adoc#resources-query-plan[`PlanResources` API] returns a query plan describing which resources a principal can access for a given action. Cerbos provides reference implementations of query plan adapters that convert that response into native query filters for your data layer, so authorization conditions are evaluated at the database level rather than in application code. Use these adapters as-is or as a starting point to build your own.
