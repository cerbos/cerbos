[#v0.10.0]
= Cerbos v0.10.0

== Highlights

This release is about making Cerbos easier to install and try out. In addition to the existing container images, binaries and Helm charts, Cerbos can now be installed using xref:deployment:systemd.adoc[Systemd service] on Linux.
