[#v0.5.0]
= Cerbos v0.5.0

== Highlights

Audit logs support in the Go SDK:: Audit log filtering capability has been added to the Go SDK to allow programmatic access to audit log streams.

`cerbosctl`:: The commands previously available under `cerbos ctl` have been moved to a separate binary named `cerbosctl` which can be downloaded separately.
